Changes from v2
CHANGED
Shortlist → Heart icon on hero. Top-right corner of the hero image. 🤍 unfilled → ❤️ filled on tap. Universal pattern — no label needed. Shows as a small heart badge on the pin on the map so user can see their saved places at a glance.
NEW
"Similar" CTA → card dismisses, map animates. Card slides down. Map zooms slightly to fit active pin + similar cluster. Similar pins glow teal with a ripple pulse — distinct from the blue added-pin glow. A banner at top reads "Similar historic sites nearby." Swipe down or tap elsewhere clears the similar state.
NEW
Dashed teal connector lines. When similar pins appear, light dashed lines connect the active pin to each similar pin — reinforces the relationship, shows geographic proximity at a glance.
CHANGED
Four pin states total. Added (blue) · Similar (teal pulse) · Reference (muted purple) · Favourited (heart badge). Each is visually distinct with no overlap. User always knows exactly what state each pin is in.
Similar places — how the LLM query works
When user taps "Similar" on Jama Masjid, fire: "Give me 4 places similar to Jama Masjid in Delhi for a cultural explorer persona — same category, nearby, worth visiting." Returns place names → match against existing map pins first (light them up) → any new places become new ghost pins. User sees instant feedback on existing pins while new ones load in.
