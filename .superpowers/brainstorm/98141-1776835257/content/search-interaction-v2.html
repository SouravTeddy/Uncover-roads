How should the search input work?

The key UX decision: how does the controlled dropdown guide users toward structured queries?

A

Smart Suggestions

AI parses partial input in real-time and surfaces structured picks. Works like autocomplete that understands intent.

🔍 museum near shinjuku
🏛 Museum near Shinjuku Station
🏛 Museum near Shinjuku Gyoen
🏛 Museum near Shinjuku (district)
B

Template Picker

Tapping search shows pre-built query templates. User picks a template and fills in the blanks. Very controlled, no ambiguity.

🔍 What are you looking for?
🗺 Place type near location
🎉 Events on a date
⭐ Best of category in this area
C

Grouped Results Dropdown

Type anything, get a grouped dropdown separating Places / Events / Areas. Results are always structured under the hood.
