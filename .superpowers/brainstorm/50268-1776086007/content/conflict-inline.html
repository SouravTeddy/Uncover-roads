How the suggestion is computed

Each answer carries a weight vector across 6 itinerary dimensions. When a conflict is detected, a scoring function evaluates all non-conflicting options against the accumulated answer weights:

score(option) = Σ weight[dimension] × alignment[option][dimension]

The highest-scoring compatible option surfaces as the suggestion. No AI — pure lookup table + dot product. The "why" line shows the two heaviest signals that drove the score.
