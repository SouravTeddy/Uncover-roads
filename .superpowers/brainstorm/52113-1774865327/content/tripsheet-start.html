B — 5th chip "Drop pin" + name field card

"Drop a pin on map" is its own chip alongside the others. When a named type is selected, a labeled card with search input expands. More structured, slightly taller.

When "Drop pin" is tapped — what happens?

P1

Sheet collapses, map enters pin-drop mode

Sheet slides down to a thin handle bar. Map is full-screen with a crosshair/target in the center and a "Confirm location" button. User pans until the pin is where they want, taps confirm, sheet comes back up.

P2

Sheet stays, tap anywhere on the map behind it

Sheet stays open but map behind it becomes interactive. User taps a spot on the visible map (above the sheet). A pin drops there. Simpler but limited to the visible map strip.
